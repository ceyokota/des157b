Feasibility Report

Features

1. Interactive Story Quiz (Branching Narrative) Feature: Users move through 8–10 scenario-based questions with multiple choices. Each choice updates hidden stats.

2. Scoring System Feature: Based on choosing different options, each choice updates the variables: career, money, health, relationships, freedom

3. Stats Compilation Feature: After quiz completion, display a visual summary of the five stats. This may use some data visualization libraries.

4. Final Results Visualization (Future Projection Screen) Feature: After completing the quiz, the project will project the users future based off of the priorities they displayed. I am thinking of using AI to display a very personalized and unique end message and future based off of all of the results, but I will need to check on the feasibility of this.

Technologies

HTML, CSS, JS, Data visualization JS libraries, Possibly AI

Level of skill

I am currently skilled enough for all these features except for the Final Results screen that may use AI.
